# VORON Fidget Toy

## Overview
A fidget toy that has two printed clicky switches
and a set of smooth rolling gears that uses VORON
BOM hardware to assemble.

Credit for original spring mechanism idea goes to @kriswillcode.

## BOM
- (4) M3x12 SHCS

## Recommended print settings

- Material: ABS or PLA
- Layer height: 0.2mm
- Extrusion width: 0.4mm
- Infill percentage: 40%
- Infill type: grid, gyroid, honeycomb, triangle, or cubic
- Wall count: 4
- Solid top/bottom layers: 5

## Assembly
Refer to included manual, or:

1. Install click springs into slots on case bottom
2. Install rocker switch and click switch on posts on case bottom ends
3. Install gears on center posts of case bottom
4. Apply case top over bottom case/components
5. Secure with 4 M3x12 SHCS, threaded into plastic. Gently!
